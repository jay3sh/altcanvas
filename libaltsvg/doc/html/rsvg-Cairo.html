rsvg_handle_render_cairo_sub ()

void                rsvg_handle_render_cairo_sub        (RsvgHandle *handle,
                                                         cairo_t *cr,
                                                         const char *id);

Draws a subset of a SVG to a Cairo surface

handle : A RsvgHandle
cr : A Cairo renderer
id : An element's id within the SVG, or NULL to render the whole SVG. For example, if you have a layer called "layer1" that you wish to render, pass "#layer1" as the id.

Since 2.14
